# HUI 0.2.0 릴리즈 노트

## 개요
UI 로직·컴포넌트를 모은 패키지로 버튼/토글/드롭다운/스크롤뷰/팝업/패널/엔티티 등을 제공합니다. Unity 2021.3+에서 동작하며 TextMeshPro, DoTween, Odin Inspector 등과 연계됩니다.

## 0.2.0에서 변경된 점
- 중복 스크립트를 제거하고 런타임 구성을 49개로 재정렬했습니다.
- 패키지 버전을 0.2.0으로 상향해 HUtil/HGame과 일관성을 맞췄습니다.
- UI 카테고리별 파일 수와 샘플 분리를 README에 명시했습니다.

## 폴더 맵 (Runtime)
- Button(6), Toggle(6): 입력·상태·이펙트 버튼/토글.
- DropDown(6): 드롭다운/옵션 선택 UI.
- Scrollview(8): 리스트/무한 스크롤/셀 관리.
- Popup(7), Panel(4): 팝업·패널 표시/애니메이션.
- Entity(4): UI 엔티티/바인딩 헬퍼.
- Spinner(1): 로딩 스피너.
- 공용 기반 1개: UI 공통 베이스.
- Samples~: UI 데모 6개.

## 업그레이드/사용 가이드
- Canvas Rebuild/Repaint 비용을 최소화하도록 빈번한 UI 갱신은 배치하거나 프레임 분산을 고려합니다.
- DoTween/UniTask/오딘/TMP 의존성을 사전에 설치한 뒤 적용합니다.
- 샘플은 참고용이며 실제 빌드에서는 제외하거나 네임스페이스를 프로젝트 규칙에 맞게 정리합니다.
