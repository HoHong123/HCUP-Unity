# HCUP 1.0.2 릴리즈 노트

----

태그 링크 : https://github.com/HoHong123/HCUP-Unity/releases/tag/v1.0.2

## 개요
HCUP 1.0.0 이후의 1.0.1(어셈블리 분리) + 1.0.2(Custom Inspector 치환) 통합 릴리즈입니다.
디렉토리 구조 평탄화, 어셈블리 식별자 prefix 통일(Hohong123 → HCUP), 패키지 컨벤션 정비를 포함하는 메이저 정비 릴리즈입니다.
Unity 2021.3+ 환경을 대상으로 하며 UniTask, DOTween(선택), Odin Inspector(선택), TextMeshPro, Addressables 에 의존합니다.

## 1.0.2에서 변경된 점

----

- 어셈블리 식별자 prefix 를 HCUP.* 로 통일했습니다 (이전: Hohong123.*).
- 패키지 디렉토리 구조를 평탄화했습니다 (HoHong123/X → X). HoHong123 컨테이너 폴더가 제거되었습니다.
- 어셈블리 분리(1.0.1) — HUtil 에 묶여 있던 HCollection / HCore / HData / HDiagnosis / HInspector 5개 도메인을 별도 어셈블리로 분리했습니다.
- Custom Inspector 치환(1.0.2) — Sound/Skill/Camera/World/Game 계층 33+ 클래스를 Odin → HInspector 로 치환했습니다.
- HCollection.Odin.Editor 어셈블리 신설 — HDictionaryToOdinBridge 가 Odin 환경에서 HDictionaryDrawer 를 강제합니다.
- AssetHandler §11 헤더 형틀 정리 — Cache/Data/Load/Provider/Store/Subscription/Validation 7개 서브폴더 25 클래스 일관 적용.
- HUI.Scrollview §11 헤더 형틀 정리 — Base/IGridSpan/IRecycle/SpanningGrid 6 클래스.
- Popup/SoundManager 등 생명주기 누수 4건 수정.
- HUtil/HUI/HGame 메인 패키지 1.0.0 → 1.0.2.

## 패키지 / 어셈블리 구성

----

- 메인 UPM 패키지 3개: HUtil, HUI, HGame
- 분리 어셈블리 5개: HCollection, HCore, HData, HDiagnosis, HInspector

## 주의사항 / 마이그레이션

----

- consumer 측 .asmdef 의 references 필드에서 `Hohong123.X` 를 `HCUP.X` 로 일괄 치환해야 합니다.
- prefab/scene/asset 의 yaml 안에 박힌 `Hohong123.X::...` 또는 `..., Hohong123.X]]` 패턴도 함께 갱신해야 type lookup 이 동작합니다.
- import 경로가 `Assets/.../HoHong123/X/...` 였다면 `Assets/.../X/...` 로 갱신해야 합니다.
- package.json 의 `name`(com.hohong123.*) 과 `displayName`(Hohong123 ...) 은 UPM 식별자 호환을 위해 유지되었습니다.
- HWindows 는 별도 1.1.0 트랙에서 진행 중이며 이번 릴리즈에 포함되지 않습니다.
