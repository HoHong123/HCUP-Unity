# HGame Changelog

## [0.2.1] - 2025-11-27
### 주요 변경
- 리포지토리 이름을 **HCUP-Unity**로 갱신하고 패키지 디렉터리를 `HoHong123/HGame` 기준으로 안내했습니다.
- README에 최신 UPM 설치 URL을 추가해 새로운 저장소/폴더 구조와 동기화했습니다.

### 마이그레이션/주의
- UPM으로 설치할 때 `HCUP-Unity` 저장소와 `HoHong123/HGame` 경로를 사용하십시오.

## [0.2.0] - 2025-11-27
### 주요 변경
- 중복 스크립트를 제거하고 런타임 구성을 53개로 정리했습니다.
- 패키지 버전을 0.2.0으로 상향해 HUtil/HUI와 일관성을 확보했습니다.
- README에 카테고리별 파일 수와 업그레이드 가이드를 추가했습니다.

### 마이그레이션/주의
- 씬/상태 전환 시 비동기 흐름이 중복 호출되지 않도록 직렬화하거나 토큰을 관리하십시오.
- 샘플(`Samples~`)은 참고용이며 실제 빌드에서는 제외하거나 별도 패키지로 분리합니다.
- 입력/플레이어/스킬 관련 코드는 프로젝트 입력 시스템에 맞게 래핑해 사용하십시오.

### 검증 체크리스트
- GameModule: 상태 전환/취소/재진입 경계 테스트.
- Camera/2D: 패럴랙스·팔로우 동작을 타임스케일/해상도 변화에서 검증.
- Skill/Player: 쿨타임·충돌·인터랙션 경합 상황에서 안전성 확인.
